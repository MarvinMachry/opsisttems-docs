# Estoque de Componentes para Montagem

## Descrição
*Data de edição: 30/10/2023*

Descrição da utilização do módulo **Estoque Montagem**.

!!! info "Campos do módulo"

    ### Item
    Define o item que será utilizado como filtro para exibir os componentes relacionados.

    ### Data Lote
    Refere-se às datas de lote disponíveis na montagem que ainda não foram concluídas.  
    Cada data de lote possui componentes referentes ao lote filtrado.

    ### Comp.
    Recebe os componentes do produto filtrado com material de código **MP01**.

    ### Cor
    Define a cor do componente, podendo ser **NOGUEIRA** ou **CASTANHO**.

    ### Local
    Define o local onde o componente foi armazenado.

    ### ESQ e DIR
    Define a quantidade armazenada no local para cada lado do componente.


---

## Controle
*Data de edição: 30/10/2023*

Descrição dos campos utilizados para controle de retirada de componentes.

!!! info "Campos de controle"

    ### Esq e Dir
    Define a quantidade que será removida dos componentes selecionados.

    ### Cor
    Permite filtrar os componentes pela cor antes de realizar a remoção.


---

## Funcionalidades
*Data de edição: 30/10/2023*

Descrição da utilização dos botões do módulo **Estoque Montagem**.

!!! tip "Botões do módulo"

    ### Liberar
    Habilita os campos para permitir a entrada de um novo componente ou atualizar sua quantidade.

    ### Adicionar
    Insere os dados preenchidos.  
    Caso o componente já exista no local informado, sua quantidade será **atualizada**.

    ### Excluir
    Exclui os componentes selecionados através da **caixa de seleção**.

    ### Remov. Qtde
    Remove a quantidade especificada nos campos **ESQ** e **DIR**.
