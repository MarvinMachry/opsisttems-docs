                    <strong class="version">Funcionalidades</strong>
                    <span class="date">Data de edição: 12/03/2026</span>
                    <p><b>Descrição da utilização do módulo Manutenção.</b></p>
                    <ul>
                      <li><b>Botão Novo:</b> Adiciona nova linha em branco para novo registro.</li>
                      <li><b>Botão Manutentor:</b> Botão utilizado para inserir uma lista de manutentores dentro de um único registro, deve ser utilizado após a inserção da manutenção.</li>
                      <li><b>Botão Histórico:</b> Abre módulo com histórico das manutenções.</li>
                      <li><b>Botão Excluir:</b> Exclui a linha em foco.</li>
                      <li><b>Botão Start:</b> Inicia a manutenção, marcando como INICIADO e abrindo o módulo para inserir os manutentores que irão trabalhar.</li>
                      <li><b>Botão Pause:</b> Pausa a manutenção, marcando como PARADO e abrindo o módulo para inserir os manutentores que pararam de trabalhar. Só abrirá caso a tarefa ja esteja iniciada.</li>
                      <li><b>Botão Concluir:</b> Abre módulo para conclusão da tarefa, os manutentores restantes (que não foram cadastrados em Pause), serão cadastrados automaticamente.</li>
                    </ul>
                </div>
            </li>
            <li>
                <div>
                    <strong class="version">Dúvidas</strong>
                    <span class="date">Data de edição: 13/03/2026</span>
                    <p><b>Descrição da utilização do módulo Manutenção.</b></p>
                    <ul>
                        <li><b>A inserção de manutentores só pode ser feita em registros existentes! Verifique sempre se o simbolo de lápis NÃO está no canto esquerdo da linha em foco. </b> </li>
                        <li><b>O Pause só pode ser dado se possuir algum registro de Start sobrando. </b> </li>
                      <li><b>A ordenação é feita na seguinte forma:</b> </li>
                      <li>1º - Status (estado_atual).</li>
                      <li>2º - Ordem.</li>
                      <li>3º - Prioridade (status).</li>
                      <li>4º - Data da Solicitação.</li>
                    </ul>
