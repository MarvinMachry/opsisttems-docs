# Programação

## Descrição
*Data de edição: 17/04/2023*

Descrição da lista do módulo **Programação**.

!!! info "Ações rápidas"

    **2 cliques no campo Data**

    Abre um formulário para alteração da data.  
    A alteração também impacta outros módulos, como **Corte Filial**.  
    Se houver itens vinculados, apenas usuários do **PCP** podem editar.

    **2 cliques no campo Config**

    Permite configurar o item para o setor de **Corte** (somente Filial).

    **2 cliques no campo Descrição**

    Abre o módulo de **Status do item**, permitindo alterar dados do produto.

## Funcionalidades
*Data de edição: 26/05/2025*

Descrição da utilização do módulo **Programação**.

!!! tip "Botões do módulo"

    **Importar**  
    Importa a programação com lote do **Promob**.  
    Apenas itens que estão em lote no Promob e ainda não existem no sistema são importados.

    **Dividir**  
    Permite dividir um item em dois registros, definindo a quantidade desejada.

    **Análise**  
    Abre um formulário com a **produtividade diária de cada setor**.

    **Relatório**  
    Abre um formulário para selecionar diferentes tipos de relatórios.

    **Hist. Datas**  
    Abre o **histórico de alterações de datas** da programação.

    **Pro.+Assis**  
    Filtro para exibir apenas **assistências**.

    **Alterações**  
    Mostra alterações de projeto relacionadas ao item.

    **Unidade**  
    Permite alternar entre **Matriz** e **Filial** para itens do tipo **MTR**.

    **Alterar Data em Massa**  
    Altera a data de todos os itens **filtrados e marcados**.  
    É obrigatório aplicar ao menos um filtro:
    
    - Descrição  
    - Pedido  
    - Observação

    **Dt. C/ Diverg.**  
    Mostra divergências de datas entre **OpSisttems** e **Promob**.

    **Marcar / Desmarcar**  
    Marca ou desmarca todos os itens atualmente filtrados na tela.
