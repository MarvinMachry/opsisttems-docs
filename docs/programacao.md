                    <strong class="version">Funcionalidades</strong>
                    <span class="date">Data de edição: 26/05/2025</span>
                    <p><b>Descrição da utilização do módulo Programação.</b></p>
                    <ul>
                        <li><b>Botão Importar:</b> Importa programação com lote do Promob (importa somente o que está em lote no Promob e o lote que ainda nao tenha entrado no sistema).</li>
                        <li><b>Botão Dividir:</b> Utilizado para dividir o item em 2 registros escolhendo a quantidade necessária.</li>
                        <li><b>Botão Análise:</b> Abre um formulario onde mostra a produtividade diaria de cada setor.</li>
                        <li><b>Botão Relatório:</b> Abre um form onde é possivel selecionar o tipo de relatirio para cada necessidade.</li>
                        <li><b>Botão Hist. Datas:</b> Abre o histórico de alterações de datas na programação.</li>
                        <li><b>Botão Pro.+Assis:</b> É um filtro para isolar somente assistencias.</li>
                        <li><b>Botão Alterações:</b> Abre formulario para ver qual é a alteração de projeto.</li>
                        <li><b>Botão Unidade:</b> Utilizado para alterar entre matriz e filial o produto, servirá para itens "MTR".</li>
                        <li><b>Botão Alterar Data em Massa:</b> Alterará a data dos itens filtrados e marcados, é obrigatório filtrar o item em ao menos 1 dos campos (Descrição, Pedido, Obs.).</li>
                        <li><b>Botão Dt. C/ Diverg.:</b> Abre um formulário para visualizar datas com divergência entre OpSisttems e Promob.</li>
                        <li><b>Botão Marcar e Desm:</b> Marca/Desmarca todos os itens filtrados (aparentes) no momento.</li>
                    </ul>
